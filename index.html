📦 Installation de la Bibliothèque Edge Impulse

  1. Téléchargez la bibliothèque en cliquant sur le bouton ci-dessous :
    📥 Télécharger ei-fall_detection_m5stick-arduino-1.0.1.zip
  2. Ouvrez le logiciel Arduino IDE.
  3. Allez dans le menu Croquis > Inclure une bibliothèque > Ajouter la bibliothèque .ZIP...
  4. Sélectionnez le fichier .zip que vous venez de télécharger.
  5. Une fois l'installation terminée, copiez le code ci-dessous et téléversez-le sur votre M5StickC.
